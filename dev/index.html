StateSpaceRealizables.ProperStateSpace — Type
ProperStateSpace{E,T,TA,TB,TC}

Type for representing proper state-space models, i.e:

\[dx = A x + B du, y = C x,\]

where d is a difference operator if E<:DiscreteTE, or a differential if E<:ContinuousTE.

source
StateSpaceRealizables.StateSpace — Type
StateSpace{E,T,TA,TB,TC,TD}

Type for representing state-space models, i.e:

\[dx = A x + B du, y = C x + D u,\]

where d is a difference operator if E<:DiscreteTE, or a differential if E<:ContinuousTE.
